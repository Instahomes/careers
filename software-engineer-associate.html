Software Engineer Associate

We are looking for aspiring software engineers to sharpen their skills as a developer at a budding proptech startup.

We are looking for aspiring software engineers to sharpen their skills as a developer at a budding proptech startup. As a Software Engineer Associate, you will be responsible for building and optimizing the frontend of various components of the product. At Instahomes, you will be able to pick up the best practices for working on a codebase and developing with a team, while also getting the chance to make ripples in the real estate industry. This is a fully remote position with highly flexible hours.

Responsibilities

  • Work alongside the CTO and CIO to further improve the frontend components of the current web application
  • Implement unit and integration testing on the system
  • You will likely work with JSX and CSS
  • You will build experimental prototypes.
  • Provide insight on issues (e.g. performance issues) that you may come across on the web application

Qualifications

  • Experience using React (or any modern JS framework)
  • Experience with implementing testing on any software before
  • Proficiency in Git (or at the very least, willing to learn!)
  • Strong analytical and problem-solving skills

Plus (but not required!)

  • Experience with MVC frameworks such as Django or Ruby on Rails
  • Experience with cloud systems such as AWS

Interested in applying?

Send your CV to instahomes.ph@gmail.com with the subject [Full Name]_[Position] (e.g. Juan Dela Cruz_Software Engineer Associate), and we'll get back to you shortly after!
